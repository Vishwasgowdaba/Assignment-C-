# C# Console Applications

This repository contains two simple **C# Console Applications** that demonstrate fundamental programming concepts such as user input, loops, variables, and basic logic building.

---

# 📌 Projects Included

1. Student Grade Calculator
2. Fibonacci Series Generator

---

# 1️⃣ Student Grade Calculator

## 📖 Overview

The **Student Grade Calculator** is a C# console application that allows users to enter details for multiple students and calculates their grades based on the marks entered.

This project demonstrates basic **C# programming concepts**, including:

* Classes and Objects
* User Input
* Loops
* Switch Expressions
* Console Output

---

## ⚙️ Features

* Accepts details for multiple students
* Stores student information such as **Name, Age, and Marks**
* Calculates grade based on marks
* Displays student details along with their grade

---

## 📊 Grade Criteria

| Marks        | Grade |
| ------------ | ----- |
| 90 and above | A     |
| 75 – 89      | B     |
| 50 – 74      | C     |
| Below 50     | Fail  |

---

## 🔄 How the Program Works

1. The user enters the **number of students**.
2. The program runs a **loop** to collect details for each student.
3. The program calculates the **grade using a switch expression**.
4. The program displays the **student information and grade**.

# 2️⃣ Fibonacci Series Generator

## 📖 Overview

The **Fibonacci Series Generator** is a simple C# console application that generates and displays the Fibonacci series based on the number of terms entered by the user.

The **Fibonacci series** is a sequence of numbers where each number is the **sum of the previous two numbers**.

Example:

0 1 1 2 3 5 8 13 ...

Formula:

Next Number = Previous Number + Current Number

---

## ⚙️ Features

* Accepts **number of terms** from the user
* Generates the **Fibonacci sequence**
* Displays the series in the console
* Demonstrates the use of **loops and variables in C#**

---

## 🔄 How the Program Works

1. The user enters the **number of terms**.
2. The program initializes the first two numbers:

   * 0
   * 1
3. A **loop** is used to calculate the next numbers.
4. Each number is printed to the console.
